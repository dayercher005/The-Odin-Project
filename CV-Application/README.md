# CV-Application

Live Preview: https://cv-application-generator-m7i3ypmsu-dayercher005s-projects.vercel.app/


## How to Run

1. Clone the repository:
```bash
git clone 
```

2. Install dependencies
```bash
npm install
```

3. Start Development Server:
```bash
npm run dev
```

4. Open `http://localHost:5173` in your browser
